- Stack

介紹

  • 堆疊 或 棧
  • 資料遵從 LIFO (Last-In-First-Out)「先進後出」的原則
  • 想像你正在放一疊書進去一個箱子，當你往這個箱子裡上放一本新書時，你只能放在最頂端。而要拿出一本書時也只能從最頂端拿，這個箱子就是 Stack
  • 可由 Array 或 Linked-List 實作
  • 支援3個操作
Operation Time Complexty Description
Peek() 查看最頂端的元素
Push() 從最頂端新增一個元素
Pop() 從最頂端移除一個元素
